# job-tracking-system
ASP.NET MVC 5 - İş Takip Sistemi... Bu web uygulaması, yönetici ve çalışan rollerine sahip kullanıcıların iş süreçlerini yönetmesini sağlar. Yöneticiler personellere görev atayabilir, iş durumlarını takip edebilir ve tüm kayıtları listeleyebilir. Çalışanlar ise kendilerine atanan işleri görüntüleyip tamamlanma durumlarını güncelleyebilir.
